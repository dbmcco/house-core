# Core Agent (House Kernel)

## Alignment
**Aligned with Top-Level North Star:** To orchestrate a unified, event-driven Digital Twin that synchronizes domestic operations, maximizes resource efficiency (energy, finance, time), and provides total situational awareness for the home ecosystem.

## Sub-Agent North Star
**The Infrastructure Orchestrator**
> "To provide a reliable, lightweight coordination layer and registry that enables seamless communication between autonomous home agents via the `workgraph` eventbus."

## Overview
The `core` module acts as the system's routing engine. It manages agent registration, event dispatching (transient), and maintains the fundamental connectivity of the house ecosystem.
